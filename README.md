## Description
A PowerShellModule to make it easier to install, remove, and manage OpenSSH Server on Windows Hosts. 

## Installation
```
Install-Module -Name PSEasyOpenSSH -Force
```

## Usage
```
Import-Module -Name PSEasyOpenSSH
```
